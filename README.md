# Luxerio Support Intake

POS and shift management intake forms for Luxerio Support.

## Deployment

This site is intended to run on Cloudflare Pages.

- Static entry: `index.html`
- POS form: `pos.html`
- Shift form: `shift.html`
- Submission endpoint: `functions/submit.js`

Responses are submitted to the existing Google Form and linked response spreadsheet.
