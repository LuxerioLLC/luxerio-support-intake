# Luxerio Support LINE運用メモ

## アカウントの役割

POS・決済、シフト管理、補助金・助成金に関する事前ヒアリングと個別案内の窓口。
「導入受付」ではなく、正式案内前の状況把握と相談受付として運用する。

## プロフィール設定

アカウント名:
`Luxerio Support`

ステータスメッセージ:
`POS/シフト/補助金窓口`

プロフィール背景:
`line-profile-cover-luxerio.png`

プロフィール表示:
投稿運用を開始するまでは「最近の投稿」を非表示にする。
空のセクションが見えると初回接点の信頼感が下がるため。

ステータスメッセージ公開時の注意:
LINE管理画面上で「今後1時間はステータスメッセージを変更できなくなります」と表示されるため、公開前に文言を確認する。

説明文:

```
Luxerio Supportは、POS・決済、シフト管理、補助金・助成金に関するご案内窓口です。
現在の運用状況を確認のうえ、担当者より個別にご案内します。
```

リンク:

| 表示名 | URL |

| POS・決済フォーム | `https://luxerio-support-intake-preview.pages.dev/pos.html` |
| シフト管理フォーム | `https://luxerio-support-intake-preview.pages.dev/shift.html` |

## 推奨あいさつメッセージ

現在のあいさつメッセージに「導入相談」「POS相談」「シフト管理相談」「入力目安1〜2分」が残っている場合は、補助金フォーム追加後の内容と矛盾するため、下記へ差し替える。

```
友だち追加ありがとうございます。
Luxerio Supportです。

POS・決済、シフト管理、補助金・助成金について、現在の状況を確認のうえ担当者よりご案内します。

下のメニューから該当フォームを選択してください。
フォーム送信後、LINEで会社名・屋号をお送りいただくと確認がスムーズです。

通常1営業日以内に担当者よりご連絡します。
https://luxerio-support-intake-preview.pages.dev/
```

## 営業・担当者の一次対応

フォーム送信後にLINEで会社名・屋号が届いたら、スプレッドシートの会社名と照合する。

推奨タグ:

- `POS`
- `Shift`
- `補助金`
- `要返信`
- `返信済`

無料プランでチャットタグ数に制限がある場合は、優先して `要返信` を付ける。

## 初回返信テンプレート

フォームのご入力ありがとうございます。
内容を確認しました。

より正確にご案内するため、確認事項がある場合はこちらのLINEでご連絡します。
担当者より通常1営業日以内にご案内しますので、少々お待ちください。

## 追加確認が必要な場合

追加質問は最大2点までに絞る。

POS・決済:

- 現在の決済手数料が不明な場合、分かる範囲で明細・契約画面を確認できるか
- 複数店舗の場合、POSや決済会社が店舗ごとに異なるか

シフト管理:

- シフト希望の回収方法
- 勤怠・給与連携の必要有無
