# Luxerio Support LINE運用メモ

## アカウントの役割

POS・決済、シフト管理に関する事前ヒアリングと個別案内の窓口。
「導入受付」ではなく、正式案内前の状況把握と相談受付として運用する。

## 推奨あいさつメッセージ

友だち追加ありがとうございます。
Luxerio Supportです。

POS・決済、シフト管理について、現在の運用状況を確認のうえ担当者よりご案内します。

1. 下のメニューから該当フォームを選択
2. フォーム送信後、LINEで会社名・屋号を送信
3. 内容確認後、通常1営業日以内に担当者よりご連絡

補助金・助成金について確認したい場合は、メニューの「補助金・相談」をタップしてください。

## 営業・担当者の一次対応

フォーム送信後にLINEで会社名・屋号が届いたら、スプレッドシートの会社名と照合する。

推奨タグ:

- `POS`
- `Shift`
- `補助金`
- `要返信`
- `返信済`

無料プランでチャットタグ数に制限がある場合は、優先して `要返信` を付ける。

## 初回返信テンプレート

フォームのご入力ありがとうございます。
内容を確認しました。

より正確にご案内するため、確認事項がある場合はこちらのLINEでご連絡します。
担当者より通常1営業日以内にご案内しますので、少々お待ちください。

## 追加確認が必要な場合

追加質問は最大2点までに絞る。

POS・決済:

- 現在の決済手数料が不明な場合、分かる範囲で明細・契約画面を確認できるか
- 複数店舗の場合、POSや決済会社が店舗ごとに異なるか

シフト管理:

- シフト希望の回収方法
- 勤怠・給与連携の必要有無
